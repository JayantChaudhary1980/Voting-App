# Voting App Backend

A simple backend API built while learning Node.js, Express, and MongoDB.

## Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication

## Features
- User registration & login
- JWT-based authentication
- Candidate management
- Voting functionality

## Folder Structure
- models/       → Mongoose schemas
- routes/       → API routes
- server.js     → Entry point
- db.js         → Database connection

## Setup
1. Clone the repo
2. Run `npm install`
3. Create a `.env` file
4. Start server with `npm start`
